Introduction

HTML stands for HyperText Markup Language.

It is the standard language for creating web pages.

<html></html>
  • Easy to learn
  • Works in all browsers
HTML Elements

HTML elements are the building blocks of webpages.

Most elements have opening and closing tags.

<p>Hello</p>
  • Paragraph
  • Heading
Attributes

Attributes provide additional information.

They are written inside opening tags.

<a href="https://example.com">Link</a>
  • href
  • src
Forms

Forms are used to collect user input.

They contain fields such as text boxes and buttons.

<form></form>
  • Input
  • Button
Semantic HTML

Semantic HTML gives meaning to content.

It improves accessibility and SEO.

<article></article> <section></section>

  • section
  • header
